\# AI Career Guidance System



An AI-powered career counseling platform designed for NGO students.



\## Features



\- 50 structured career dataset

\- Weighted multi-factor career matching engine

\- Financial constraint awareness

\- Rural / Urban suitability filtering

\- AI-generated personalized roadmap

\- Multilingual support (English / Hindi)

\- Professional dark UI



\## Tech Stack



Frontend: React

Backend: Node.js + Express

AI: OpenRouter API

Architecture: Hybrid (Rule-based + LLM)



\## How It Works



1\. Student fills profile

2\. Rule-based engine scores careers

3\. AI generates explanation \& roadmap

4\. System outputs realistic guidance
